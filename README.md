# Dados dos Sistemas de Alertas Hidrológicos do SGB

Nesta página, são apresentados resultados de análises de dados aplicadas a de dados de estações fluviométricas pertencentes à Rede Hidrometeorológica Nacional e que fazem parte dos Sistemas de Alertas Hidrológicos do SGB / CPMR (Serviço Geológico do Brasil).

São apresentados os seguintes dados:
- Gráficos com os últimos dados de estações de monnitoramento fluviométrico;
- Previsões feitas em estações específicas;
- Gráficos sazonais, que incluem dados de 2023 diante de um fundo que representa a variação sazonal dos dados das estações;
- Tabelas com os últimos dados de algumas bacias brasileiras;
